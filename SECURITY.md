# Security Policy

## What This Repo Contains

This repository contains AI skill definitions (Markdown files) that shape how Claude responds on Gene Boyle's behalf. These files contain:

- Business processes and procedures
- Communication templates
- Market analysis frameworks
- Competitive intelligence

## What This Repo Must NEVER Contain

- **GitHub Personal Access Tokens** (PATs) or any API keys
- **Client names, case numbers, or identifiable estate details**
- **Attorney names tied to specific cases** (general relationship references are OK)
- **Commission rates or fee structures**
- **Financial account numbers or investment positions**
- **Passwords or login credentials**

## If a Secret Is Committed

1. **Revoke it immediately** — even before removing from git history
2. Contact Kelly at kellylynnboyle@gmail.com
3. Use `git filter-branch` or BFG Repo Cleaner to remove from history
4. Force push the cleaned history

Remember: deleting a file in a new commit does NOT remove it from git history.
