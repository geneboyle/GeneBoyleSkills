## CMA Structure for Probate Properties

Probate CMAs differ from standard residential CMAs because the audience includes attorneys and courts, not just sellers.

### Required Components

1. **Subject Property Overview** — Address, beds/baths/sqft, lot size, year built, condition assessment
2. **Comparable Sales** (minimum 5, sold within 6 months, same zip or adjacent)
   - Price, DOM, price/sqft, condition, sale type (standard, probate, trust, REO)
   - Adjustments for condition, lot size, location, upgrades
3. **Active/Pending Listings** — Current competition the property will face
4. **Probate-Specific Adjustments**
   - As-is condition discount (typically 5-15% below retail if no repairs)
   - Court confirmation timeline premium/discount
   - Estate sale stigma factor (minimal in OC luxury market)
5. **Recommended List Price** — With justification tied to comps and court requirements
6. **90% Floor Calculation** — If court confirmation required: `Appraised Value × 0.90`
7. **Marketing Plan** — How Gene will position and expose the property
8. **Timeline Estimate** — From listing to projected close

### Pricing Framework

```
Probate Property Valuation Logic:

Step 1: Establish retail value (if property were in market-ready condition)
  → Based on sold comps, adjusted for location/condition/features

Step 2: Apply condition adjustment
  → Deferred maintenance: -5% to -15%
  → Major systems issues: -10% to -25%
  → Cosmetic only: -0% to -5%

Step 3: Consider sale type
  → Full IAEA (no court confirmation): Price like standard sale
  → Court confirmation required: Price at 95-100% of appraised value
    (attract competitive offers, let overbid process maximize price)

Step 4: Verify against 90% floor (if court confirmation)
  → List price must generate offers ≥ 90% of appraised value

Gene's recommended list price = Adjusted value × positioning factor
  Positioning factor:
    0.95 — generate multiple offers quickly (tight timeline)
    1.00 — balanced approach (standard recommendation)
    1.05 — test the market (luxury properties, strong demand areas)
```

## Listing Presentation Deck Outline

When Gene presents to a PR and/or attorney, the deck should follow this structure:

1. **About Gene** — 100% probate specialization, CPRES, 20+ years, BHHS backing
2. **Understanding Your Situation** — Acknowledge the loss, demonstrate process knowledge
3. **The Property** — Photos, key features, condition assessment
4. **Market Analysis** — Comps, trends, pricing recommendation (the CMA)
5. **Gene's Marketing Plan** — MLS, BHHS network, attorney network, photography, staging
6. **The Probate Sale Process** — Visual timeline showing each phase
7. **What Makes This Different** — Probate-specific expertise (court, overbids, fiduciary)
8. **Gene's Track Record** — 12% above valuations, client testimonials
9. **Your Investment** — Commission structure (discussed privately, not in written materials)
10. **Next Steps** — Clear actions for moving forward

## Property Types Gene Encounters

| Type | Typical Condition | Strategy Notes |
|------|------------------|----------------|
| Luxury coastal (CDM, Newport, Laguna) | Often well-maintained, high value | BHHS luxury marketing, international exposure |
| Mid-range suburban (Irvine, Costa Mesa) | Variable — may need updates | Strong local comp pool, price competitively |
| Older estate (60s-80s construction) | Deferred maintenance common | As-is marketing to investors + retail buyers |
| Multi-unit / income property | Tenant-occupied complications | Address occupancy issues with attorney first |
| Vacant land | Rare in OC probate | Development potential analysis needed |

## Presentation Do's and Don'ts

**Do:**
- Use actual sold comps from the last 6 months (search every time)
- Show the 90% floor calculation transparently
- Include a visual timeline
- Reference Gene's attorney network as a marketing advantage
- Use professional photography (never phone photos in presentations)

**Don't:**
- Overpromise on price (fiduciary duty requires honesty)
- Include commission rate in written materials
- Use industry jargon without defining it
- Compare Gene to specific competitors by name
- Rush the family — let the presentation breathe
