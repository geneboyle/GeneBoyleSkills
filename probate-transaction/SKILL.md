# Probate Transaction Playbook

> Load `avatar/SKILL.md` first for Gene's voice and operating principles.

## Transaction Phases

Every probate RE transaction moves through 8 phases. Gene's value is knowing exactly what happens at each stage and keeping all parties aligned.

### Phase 1: Referral & Intake

**Trigger:** Attorney calls, family finds JustCallGene.com, or existing client refers.

**Gene's process:**
1. Acknowledge the loss first — "I'm sorry for what your family is going through"
2. Ask key qualifying questions:
   - Has probate been filed? (If not, refer to an attorney first)
   - Who is the Personal Representative (PR) / Executor?
   - Is there a will? Trust? Intestate?
   - IAEA authority granted? Full or limited?
   - Property location and approximate condition?
   - Any liens, encumbrances, or title issues known?
   - Timeline pressure? (Creditor claims, family disputes, carrying costs)
3. Schedule property visit within 48 hours if possible
4. Send follow-up email within 2 hours summarizing the call and next steps

**Output template:** See `references/intake-checklist.md`

### Phase 2: Property Assessment

**Gene's process:**
1. Visit property — assess condition, needed repairs, presentation potential
2. Pull preliminary comps (active, pending, sold within 6 months, same zip)
3. Note any condition issues that affect court-appraised value
4. Photograph extensively (interior, exterior, neighborhood context)
5. Identify value-add opportunities (light staging, cleaning, minor repairs)
6. Prepare preliminary valuation range for the PR and attorney

**Key consideration:** The probate referee will set the official appraised value. Gene's preliminary assessment helps the PR understand what to expect, but Gene never represents his number as the official appraisal.

### Phase 3: Court Petition Support

**Gene does NOT file legal documents.** That's the attorney's role. But Gene supports by:

1. Providing market data and preliminary valuation to inform the petition
2. Coordinating with the probate referee if the attorney requests
3. Preparing a market analysis that supports the proposed listing strategy
4. Being available to answer the court's questions about real property value if needed

**Key CA Probate Code references:**
- **§ 8800-8843** — Inventory and appraisal requirements
- **§ 10300-10316** — Sale of real property provisions
- **§ 10500-10503** — Court confirmation requirements (when needed)
- **IAEA (§ 10400-10416)** — Independent administration authority

### Phase 4: Listing Strategy

**Two tracks depending on IAEA authority:**

**Full IAEA Authority (most common in modern estates):**
- PR can sell without court confirmation
- Standard MLS listing process
- Must still get fair market value (fiduciary duty)
- Gene recommends listing at or slightly above appraised value to demonstrate fiduciary care
- Marketing period: typically 2-4 weeks on market

**Limited or No IAEA Authority (court confirmation required):**
- Minimum offer: 90% of appraised value (Cal. Prob. Code § 10309(a)(3))
- Appraisal must be within 1 year of confirmation hearing date
- Sale is subject to overbidding at court hearing
- Gene prices to attract competitive offers knowing overbid is possible
- Marketing period: list immediately, build interest before hearing date

**Pricing framework:**
```
Court-confirmed sale minimum = Appraised Value × 0.90
Gene's recommended list price = Appraised Value × 0.95–1.05
  (attracts multiple offers, demonstrates fiduciary duty, maximizes overbid competition)
```

### Phase 5: Marketing Period

**Gene's marketing toolkit:**
- MLS listing with probate-specific disclosures
- BHHS luxury network (for $2M+ properties)
- Attorney network notification (50+ firms = potential buyer referrals)
- Professional photography + staging recommendations
- Targeted outreach to investors who buy probate properties (for fixer/as-is)
- JustCallGene.com featured listing

**Disclosures specific to probate:**
- Property sold in "as-is" condition (typical for probate)
- Sale subject to court confirmation (if applicable)
- PR has no personal knowledge of property condition (standard disclaimer)
- Buyer to investigate all matters of concern independently

### Phase 6: Offer Review

**Gene evaluates offers on these criteria:**
1. Price vs. appraised value (must meet 90% floor if court-confirmed)
2. Buyer financing strength (cash > conventional > FHA for probate)
3. Contingency terms (fewer = stronger for court)
4. Closing timeline (must align with court calendar if confirmation needed)
5. Buyer's ability to attend court hearing if overbid (for confirmed sales)
6. Deposit amount (shows seriousness to the court)

**Gene's recommendation format to the PR/attorney:**
- Present all offers side-by-side
- Highlight pros/cons of each
- Make a recommendation but emphasize: "The decision is yours as fiduciary"
- Document the recommendation in writing

### Phase 7: Court Confirmation (If Required)

**Pre-hearing preparation:**
1. Confirm hearing date and department with the court clerk
2. Prepare the PR for what happens in the courtroom
3. Brief the buyer on overbid possibility and process
4. Ensure all parties know the required deposits

**Overbid rules (CA Probate Code):**
```
First overbid minimum = Original offer price + 5% of first $10,000 + 10% of remainder

Example on $1,000,000 offer:
  5% of $10,000 = $500
  10% of $990,000 = $99,000
  Minimum first overbid = $1,000,000 + $500 + $99,000 = $1,099,500

Subsequent overbids: Must exceed previous bid by the greater of $5,000 or the
  increment set by the court at the hearing
```

**At the hearing:**
- Gene attends to support the PR and answer property questions
- If overbidding occurs, Gene manages the room — keeps the PR calm, tracks bids
- Final sale is confirmed by the judge
- Gene celebrates quietly — this is a family's loss being resolved, not a victory lap

### Phase 8: Close

**Post-confirmation to close:**
1. Open escrow immediately (or confirm existing escrow)
2. Coordinate with title company on probate-specific title requirements
3. Manage any repairs or credits negotiated
4. Keep PR and attorney updated weekly (minimum)
5. Attend final walkthrough with buyer (probate properties often have surprises)
6. Close and confirm distribution instructions with the attorney

**Post-close:**
- Thank the family personally (handwritten note, not just email)
- Thank the referring attorney (this is the relationship that generates future business)
- Request testimonial if appropriate (timing matters — not too soon)
- Update CRM with case outcome and referral source

## Red Flags Gene Watches For

- PR who wants to sell below appraised value (potential breach of fiduciary duty)
- Family disputes among heirs (Gene stays neutral, defers to the attorney)
- Property with environmental issues (mold, lead, asbestos — common in older estates)
- Outstanding liens or property tax delinquencies
- Occupants who won't vacate (tenant or family member in the property)
- Multiple creditor claims that may exceed estate value
- PR who is also a buyer or has a conflict of interest
